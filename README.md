## 🌍 About the Project

**Hasiru Usiru** is a community-driven platform designed to track and encourage environmental conservation efforts. Whether it's organizing local tree-planting drives, mapping urban greenery, or providing real-time air quality metrics, this project aims to bridge the gap between technology and nature conservation. 

Our goal is to make environmental activism accessible, trackable, and engaging for everyone.

---

## ✨ Key Features

* **🗺️ Interactive Tree Mapping:** Users can pin locations of newly planted trees, upload photos, and track their growth over time.
* **📊 Air Quality Dashboard:** Integration with local APIs to display real-time AQI (Air Quality Index) in your area.
* **📅 Community Events:** A built-in calendar for organizing and joining local clean-up drives and afforestation events.
* **🏆 Leaderboard & Gamification:** Earn "Green Points" for eco-friendly activities and compete with local communities to climb the leaderboard.
* **📱 Responsive Design:** Fully accessible on both desktop and mobile devices.

---

## 💻 Tech Stack

* **Frontend:** React.js / Next.js, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB / PostgreSQL
* **Mapping:** Mapbox API / Leaflet
* **Deployment:** Vercel (Frontend), Heroku / Render (Backend)
